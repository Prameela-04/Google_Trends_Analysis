# Google_Trends_Analyis
Analyzes and compares Google Trends search interest for Cloud Computing, Data Science, and Machine Learning using Python, PyTrends, and Matplotlib. Visualizes keyword popularity trends over time through interactive data analysis.
# Google Trends Keyword Analysis

This project uses Google Trends data to compare the popularity of Cloud Computing, Data Science, and Machine Learning over time. The analysis is performed using Python and the PyTrends library, with results visualized using Matplotlib.

## Features
- Fetches real-time Google Trends data
- Compares multiple keywords
- Generates trend visualization graphs
- Simple and easy-to-understand implementation

## Technologies Used
- Python
- PyTrends
- Pandas
- Matplotlib
- Google Colab

## Output
The project generates a line graph showing the search interest trends of selected keywords over a specified time period.
